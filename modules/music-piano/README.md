# Project Mentor Piano MVP

This is the first runnable application module for Project Mentor.

## What it does

- Connects to a MIDI keyboard
- Displays a digital piano
- Lights up keys as you play
- Detects chords live
- Lets you choose a target chord
- Shows green/red feedback
- Includes simple progression practice

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
python src/app.py
```

## First test

1. Plug in your MIDI keyboard.
2. Run the app.
3. Click Refresh.
4. Choose your MIDI input.
5. Click Connect.
6. Try playing:

- C E G
- C E G Bb
- D F A C
- G B D F
