# Knowledge Graph

The Knowledge Graph maps how concepts connect.

The mentor should not teach isolated lessons. It should understand prerequisites, dependencies, and next steps.

## Music Example

```text
Sound
└── Rhythm
    ├── Pulse
    ├── Meter
    ├── Subdivision
    └── Syncopation

Pitch
├── Notes
├── Intervals
├── Scales
│   ├── Major Scale
│   ├── Minor Scale
│   └── Modes
├── Triads
│   ├── Major
│   ├── Minor
│   ├── Diminished
│   └── Augmented
├── Seventh Chords
│   ├── Major 7
│   ├── Minor 7
│   ├── Dominant 7
│   └── Half Diminished
└── Progressions
    ├── I-IV-V
    ├── ii-V-I
    ├── I-V-vi-IV
    └── 12-Bar Blues
```

## Why This Matters

If a learner struggles with ii-V-I, the mentor should reason backward.

Maybe the problem is not ii-V-I.

Maybe the learner does not understand seventh chords, chord function, scale degrees, intervals, or major/minor quality.

The graph helps the mentor teach the real missing concept.
