# AI Piano Teacher Module

This is Module One for Project Mentor.

The goal is to prove that an AI mentor can observe live performance, understand mistakes, and teach music concepts in real time.

## Current Prototype Goals

- MIDI input
- Digital keyboard UI
- Chord detection
- Target chord practice
- Progression training
- Green/red note feedback
- Style-aware critique

## Next Build

The next serious build should focus on:

1. Practice memory
2. Better chord engine
3. Rhythm scoring
4. “Why?” button
5. MusicXML import
